# Tech Stack Recommendations

After detecting the framework in Step 1, recommend relevant plugins, MCP servers, and skills.
Present as: "Based on your tech stack, I recommend installing these:" with install commands.

## Laravel

### Must Have
| Tool | Type | Install | Why |
|------|------|---------|-----|
| **laravel-boost** | MCP Server | `claude mcp add laravel-boost -- npx @nicholasgriffintn/laravel-boost-mcp` | DB schema, queries, error logs, 17K+ docs. Built for Laravel. |
| **context7** | Plugin | `/plugin install context7` | Live Laravel/Filament docs — never use stale training data |
| **php-lsp** | Plugin | `/plugin install php-lsp` | PHP language server — autocomplete, go-to-definition |

### Recommended
| Tool | Type | Install | Why |
|------|------|---------|-----|
| **playwright** | Plugin | `/plugin install playwright` | E2E browser testing for Livewire/Filament pages |
| **chrome-devtools-mcp** | Plugin | `/plugin install chrome-devtools-mcp` | Debug frontend, inspect network, Lighthouse audits |
| **coderabbit** | Plugin | `/plugin install coderabbit` | AI code review on PRs |
| **feature-dev** | Plugin | `/plugin install feature-dev` | Guided feature development with architecture focus |

### If Using Filament
| Tool | Type | Why |
|------|------|-----|
| **laravel-boost** | Already listed | Filament resource schema inspection |
| Custom Filament rules | Generated by buddyx-forge | Section → Tabs → Tab → Grid, authorize(), etc. |
